# Code of Conduct

## Our Standards

- Be direct and honest
- Be fair to the people and companies you write about
- Cite evidence; don't make claims you can't support
- Acknowledge when you're wrong
- No harassment, hate speech, or personal attacks

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, and discussions.

## Enforcement

Violations may result in removal of content and/or blocking from the repository. Open an issue to report a conduct concern.
