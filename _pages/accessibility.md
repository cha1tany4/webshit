**Last updated:** March 2026

Webshit is committed to WCAG 2.1 Level AA compliance, with selective AAA targets.

## Our Standards

- **Contrast ratio:** Minimum 7:1 for body text (AAA)
- **Touch targets:** Minimum 44×44px
- **Keyboard navigation:** Full site navigable without a mouse
- **Reduced motion:** Respects `prefers-reduced-motion`
- **Semantic HTML:** Proper heading hierarchy, landmark regions

## Third-Party Scripts

The only third-party script on this site is [GoatCounter](https://www.goatcounter.com/) analytics. It loads with the `async` attribute and has no effect on page rendering or keyboard/screen reader access. The site is fully functional with JavaScript disabled — GoatCounter simply won't load, which is fine.

## Known Issues

None currently documented. If you find an accessibility issue, please [report it](https://github.com/webshitorg/webshit/issues).

## Testing

This site is tested using:
- Chrome DevTools Lighthouse (Accessibility)
- Axe DevTools
- Manual keyboard navigation

## Feedback

If you encounter any accessibility barriers, [open a GitHub issue](https://github.com/webshitorg/webshit/issues).
