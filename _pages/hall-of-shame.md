The most egregious examples of web failures, curated for posterity.

*This section is being built. [Contribute a nomination.](/contribute/)*

## Criteria for Entry

To make the Hall of Shame, a web failure must be:

1. **Documented** — Evidence exists (screenshots, reports, source code)
2. **Impactful** — Affected a significant number of users
3. **Avoidable** — Better choices were clearly possible at the time
4. **Instructive** — There's something to learn from it

## Nominations

Open a GitHub issue with the label `hall-of-shame` to nominate a candidate.
