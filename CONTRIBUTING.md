# Contributing to Webshit

Webshit is a community archive. We welcome contributions from anyone.

## Ways to Contribute

- **Write a post** — Submit a short or essay via pull request
- **Report an error** — Open an issue if something is factually wrong
- **Suggest a topic** — Open an issue with the `suggestion` label
- **Fix accessibility** — PRs improving accessibility are always welcome

## Post Guidelines

See [/contribute/](/contribute/) on the site for full writing guidelines.

Quick version:
- Shorts: up to 300 words, direct observation
- Essays: 1500–3000 words, with context, causes, and solutions
- Always cite sources
- Always suggest better approaches

## Pull Request Process

1. Fork the repository
2. Create a branch: `git checkout -b post/your-post-title`
3. Add yourself to `_data/authors.yml` if this is your first post
4. Write your post in `_posts/YYYY-MM-DD-title.md` using your key in the `author` field
5. Test locally: `bundle exec jekyll serve`
6. Submit a pull request with a brief description
